# Security Policy

## Supported versions

Only the latest release of OptiLab Core receives security fixes.
Older releases are not patched.

| Version | Supported |
|---------|-----------|
| 1.1.x (latest) | ✅ |
| < 1.1.0 | ❌ |

## Reporting a vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.

Send a description of the issue to:

**info@lanesaudio.com**

Include:

- The OptiLab Core version and format (JSFX or Winamp DSP DLL).
- The host application and Windows version.
- A brief description of the issue and, where possible, a minimal reproducing
  case.

You can expect an acknowledgement within a few business days. If a fix is
warranted, a patched release will be prepared and credit offered in the
changelog unless you prefer to remain anonymous.
